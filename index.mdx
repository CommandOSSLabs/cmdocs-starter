# Welcome to My Docs

This documentation site was generated with [cmdocs](https://github.com/CommandOSSLabs/cmdocs) — an open-source, self-hosted documentation platform powered by [Fumadocs](https://fumadocs.dev) and [Next.js](https://nextjs.org).

## Features

<Cards>
  <Card title="MDX Content" description="Write documentation in MDX — Markdown with JSX components, auto-imported for you." />
  <Card title="Customizable Themes" description="Configure colors, dark mode, layout, and more via docs.json." />
  <Card title="Built-in Search" description="Full-text search powered by Orama — works out of the box with static exports." />
  <Card title="Developer-First" description="CLI workflow with hot-reload, static export, and Git-based deployments." />
</Cards>

## How It Works

<Steps>
  <Step>
    **Write** — Author your documentation in MDX files. Use Fumadocs components like Callout, Card, Tabs, and more.
  </Step>
  <Step>
    **Configure** — Edit `docs.json` to customize your site name, theme, and navigation structure.
  </Step>
  <Step>
    **Deploy** — Run `cmdocs build` for a static site, or connect your Git repo to the cmdocs dashboard for automated deployments.
  </Step>
</Steps>
