# Writing Content

cmdocs uses [MDX](https://mdxjs.com/) — a format that lets you use JSX components inside Markdown documents. All Fumadocs components are auto-imported, so you never need to write import statements.

## Frontmatter

Every page starts with a YAML frontmatter block:

```yaml
---
title: Page Title
description: A brief description of this page.
---
```

| Field | Description | Required |
|-------|-------------|----------|
| `title` | Page title displayed in the heading and sidebar | Yes |
| `description` | Page description for SEO meta tags and search results | No |
| `icon` | Icon name displayed next to the page title in the sidebar | No |
| `full` | Hide the table of contents and use full-width layout | No |

## Markdown Basics

All standard Markdown features are supported:

### Text Formatting

```md
**Bold text**
*Italic text*
~~Strikethrough~~
`inline code`
```

### Links

```md
[External link](https://example.com)
[Internal link](/docs/quickstart)
```

### Lists

```md
- Unordered item
- Another item
  - Nested item

1. Ordered item
2. Another item
```

### Blockquotes

```md
> This is a blockquote.
> It can span multiple lines.
```

### Code Blocks

Use fenced code blocks with a language identifier for syntax highlighting:

````md
```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}
```
````

Supported languages include: `typescript`, `javascript`, `python`, `rust`, `go`, `bash`, `json`, `yaml`, `css`, `html`, and many more.

### Tables

```md
| Column 1 | Column 2 |
|----------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |
```

### Images

Place images in the `public/` directory and reference them with an absolute path:

```md
![Alt text](/my-image.png)
```

## JSX Components

The power of MDX is using components directly in your content. cmdocs auto-imports all Fumadocs components — just use them:

```mdx
<Callout type="info">
  This is an informational callout.
</Callout>
```

See the [Components](/docs/guides/components) page for all available components.
